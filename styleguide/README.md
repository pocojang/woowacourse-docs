# 우아한테크코스 스타일 가이드

개발자는 가능한 간결하고 읽기 쉬운 코드를 작성해야 합니다. 때문에 다른 개발자와 약속한 스타일로 코드를 작성하는 일은 중요한 과정입니다. 우아한테크코스는 이미 많은 개발자들이 고민하고 노력한 결과로 나온 스타일 가이드를 기본으로 하지만, 크루들과 함께 이 스타일 가이드를 얼마든지 개선해나갈 수 있습니다.
